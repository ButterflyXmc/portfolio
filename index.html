My Latest Projects

THE FIVE MINUTE JOURNAL

The Five Minute Journal is a web application designed to help users develop a positive mindset and nurture gratitude by taking just five minutes a day to reflect on the good things in life. The application is more than just a notebook - it's a powerful tool that encourages users to focus on the positive aspects of their day and cultivate a sense of gratitude and appreciation.

Built using Vue.js for the frontend, Python Flask for the backend, and Debeaver for the database, the Five Minute Journal provides users with an intuitive and easy-to-use platform for creating and managing their daily reflections.Additionally, the application includes authentication functionality to ensure user data privacy and security.

GOLDEN PIZZA

For my demo project, I created Golden Pizza, a website that allows users to easily browse the menu and connect with the restaurant. Some of the key features of the site include a fake contact us social media page, hours of operation, and a "Laugh with Us" page that includes cheesy pizza jokes to add a playful touch.

Using HTML and CSS, I was able to create a visually appealing and functional website that showcases my skills in web development. The project was a fun learning experience that helped me improve my HTML and CSS skills while creating something that is both fun and practical.
